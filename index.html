Arcade Game

This project required developing a Frogger-like arcade game using HTML5 Canvas. The objective of the game is to earn points by reaching the water without colliding into an enemy bug.

How to play:

- Use the up, down, left, right arrow keys to move your character.
- You lose 50 points for each collision and gain 50 points for each successful trip to the river.
- When you have no more points left, the game is over.
- You win the game by earning 1000 points.

Optimization

This project required optimizing the critical rendering path of two given websites to achieve:

- a PageSpeed Insights score of at least 90
- at least 60 fps rendering performance while scrolling

Details:

Optimization involved using build tools to minify and compress HTML, CSS, and JavaScript files. Code changes included caching reused elements to eliminate repetitive reads and writes to the DOM, preventing creation of redundant variables, and adding CSS styles to improve rendering speeds.

Neighborhood Map

A single-page application featuring a map of a favorite neighborhood, Doolin, Ireland.

Features include:

- a Google Maps view with map markers to identify popular locations
- a search function to easily discover these locations
- a listview to support simple browsing of all locations
- integration with an additional third-party API to provide information about each locations
- a responsive UI

Implementation:

This site uses the KnockoutJS framework to implement observables, event handlers, filters, and to handle other state changes. It uses the Google Maps API along with markers and infowindows, and the Wikimedia API to access Wikipedia content for each location. All API calls are made asynchronously with appropriate error handling.
